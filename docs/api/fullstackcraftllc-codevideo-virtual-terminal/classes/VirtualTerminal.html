Class VirtualTerminal

Represents a virtual terminal that can be interacted with via a series of actions. The terminal maintains a command history and a current command that can be modified. The terminal also maintains a caret position that can be moved around the current command. Finally, the terminal is also responsible for it's entire own buffer, for easy rendering in UIs.
